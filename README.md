# medical-information-system

1. The goal of this is project is to create a backend API for a simplified medical information system. The API will allow users to perform CRUD (Create, Read, Update, Delete) operations on patient records, appointments, and medical staff information.

2. The API will be built using ASP.NET Core Web API, and it will use Entity Framework Core for data access. The database will be a SQL Server database.

3. Current status is that the project is in the early stages of development. The basic structure of the API is still developing
